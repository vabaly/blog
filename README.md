# Vabaly 的博客

    致力于前端方面的原创深度好文

* 仓库的文章是草稿，正式的文章放在 Issues 中
* Commit 规范遵循以下规则
    * `create` => 创建文章
    * `edit` => 编辑文章
    * `finish` => 完成文章
    * `chore` => 项目工程的改动
    * `docs` => 修改 `README`
